# Predicao De Droga

Prevendo o tipo de droga a partir de algumas variaveis


### Features:

  Age (idade)
  
  Sex (sexo)
  
  BP (Pressão Arterial)
  
  Cholesterol (Nível de Colesterol)
  
  Na_to_K (Nível da dosagem)
  
  Drug (Target)
  
## Requisito do Projeto
95% DE PRECISÃO NA PREVISÃO NO TIPO DE DROGA
